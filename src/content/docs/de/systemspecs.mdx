Um Remote Incident Manager ausführen zu können, muss Ihr System mindestens die folgenden Anforderungen erfüllen:

<Windows>
  * Windows 7 SP1 oder höher
  * Windows Server 2012 oder höher
  * Mindestens 4 GB RAM
  * Admin-Rechte für die Erstinstallation (nicht erforderlich für automatische Updates)
</Windows>

<MacOS>
  * Intel oder Apple Silizium
  * macOS Ventura oder höher
  * Mindestens 4 GB RAM
  * Admin-Rechte für die Erstinstallation (nicht erforderlich für automatische Updates)
</MacOS>
