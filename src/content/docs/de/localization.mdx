title: Lokalisierung
---

Remote Incident Manager wird derzeit mit 6 Hauptsprachen ausgeliefert, weitere werden im Laufe der Zeit hinzugefügt. Wenn Ihr Systemgebietsschema zu den von RIM unterstützten Sprachen gehört, wird RIM die von Ihnen angegebene Sprache verwenden.

# Umschalten der Sprache

Unabhängig davon, ob Sie sich im Controller- oder im Zielmodus befinden, zeigt Remote Incident Manager am unteren Rand des Bildschirms eine Schaltfläche zur Sprachauswahl an. Wenn Sie auf diese Schaltfläche klicken, wird eine Liste mit Sprachen angezeigt. Wählen Sie die gewünschte Sprache aus, und klicken Sie dann auf die Schaltfläche "Sprache wechseln".

# Weitere Sprachen

Wir freuen uns über Beiträge von Einzelpersonen und Unternehmen zu weiteren Sprachen. Informationen darüber, wie Sie Sprachen beitragen können, finden Sie unter [Besuchen Sie das Localization Repository auf Github](https://github.com/PneumaSolutions/rim-localization)
