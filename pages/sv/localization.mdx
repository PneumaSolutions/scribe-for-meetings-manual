# Lokalisering

Remote Incident Manager levereras för närvarande med 6 större språk, och fler språk kommer att läggas till med tiden. Om ditt systemspråk finns bland de språk som stöds av RIM, kommer RIM att använda det språk du angav.

# Byta språk

Oavsett om du är i kontrollant- eller målläge visar Remote Incident Manager en språkväljarknapp längst ned på skärmen. Om du klickar på denna visas en lista över språk. Välj önskat språk och klicka sedan på knappen "Byt språk".

# ytterligare språk

Vi välkomnar bidrag med ytterligare språk från både privatpersoner och företag. För information om hur du bidrar med språk, [Besök Localization repository på Github](https://github.com/PneumaSolutions/rim-localization)
