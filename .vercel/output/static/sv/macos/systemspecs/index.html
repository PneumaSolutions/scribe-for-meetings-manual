Systemkrav

För att kunna köra Remote Incident Manager måste ditt system uppfylla eller överträffa följande krav:

  • Intel- eller Apple-kisel
  • macOS Ventura eller senare
  • Minst 4 GB RAM-minne
  • Administratörsbehörighet för den första installationen (krävs inte för automatiska uppdateringar)
