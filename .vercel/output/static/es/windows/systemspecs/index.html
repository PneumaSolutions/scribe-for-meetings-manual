Requisitos del sistema

Para ejecutar Remote Incident Manager, su sistema debe cumplir o superar los siguientes requisitos:

  • Windows 7 SP1 o posterior
  • Windows Server 2012 o posterior
  • Al menos 4 GB de RAM
  • Privilegios de administrador para la instalación inicial (no necesarios para las actualizaciones automáticas)
