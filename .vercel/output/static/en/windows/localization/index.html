Localization

Scribe for Meetings currently ships with 6 major languages, with more being added over time. If your system locale is among the supported languages in Scribe for Meetings, the user interface will use your chosen language. Note that presentation content will still be presented in the language of the original document.

Switching languages

Scribe for Meetings displays a language selector button at the bottom of the screen. Clicking this will display a list of languages. Select the desired language, and then click the “Switch language” button.
